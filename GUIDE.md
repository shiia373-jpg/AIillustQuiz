# 🌾 農場ゲーム

```
① 種を買う（ショップ🛒）
② 畑に植える（🌱 植える）
③ 育つのを待つ
④ 収穫する（🧺 全部収穫）
⑤ コインを稼いでレベルアップ！
```

## 💎 品質システム

収穫時にランダムで品質が決まります。品質が高いほど販売額が増えます！

| ランク | 倍率 | 出やすさ（基本） |
|---|---|---|
| 💎 SS | ×4.0 | 約1% |
| ⭐ S | ×2.5 | 約6% |
| ✨ A | ×1.8 | 約13% |
| 👍 B | ×1.4 | 約25% |
| 😐 C | ×1.0 | 約55% |

> 💡 家具・外装の **品質UP** ボーナスを積むほど高品質になりやすい！

| 状態 | タイミング | ボーナス |

## 💤 放置ボーナス

農場を長時間放置すると自動収穫されますが、放置時間が長いほど効率UP（最大×1.8）。
ただし手動収穫より効率は下がります。

- レベルが上がると自動収穫の効率が上がる

# 🎨 AIイラストクイズ

AIが描いたイラストを見て、お題を当てるクイズゲームです。

## ゲームの流れ

### 出題者の手順

```
① /start-quiz rounds:5（ラウンド数を指定）
② 「ラウンド1のお題を入力」ボタンを押す
③ お題を入力（例：りんご、ねこ）
④ AIがイラストを生成 → プレビューで確認
⑤ ✅ 承認 → クイズ開始！
⑥ 必要に応じてヒントを追加
⑦ 全員正解または「ギブアップ」で次のラウンドへ
```

### 回答者の手順

```
① クイズ画面の「✏️ 回答する」ボタンを押す
② ひらがなまたはカタカナで回答を入力
③ 正解するとポイント獲得！
```

---

## 🏆 ポイントシステム

| 正解順位 | ポイント |
|---|---|
| 1人目 | +3pt |
| 2人目 | +2pt |
| 3人目 | +1pt |
| 4人目以降 | +0pt |

---

## 🕹️ 出題者が使えるボタン

| ボタン | 説明 |
|---|---|
| 💡 ヒントを追加 | 回答者へのヒントを入力して追加 |
| 📋 正解者確認 | 現在の正解者とポイントを確認（非公開） |
| ⏭️ 次のラウンド | 次のお題を入力して続ける |
| 🏳️ ギブアップ | 正解を公開して次へ |
| 🛑 ゲームを止める | クイズを終了して結果を表示 |

---

## ⚠️ 注意事項

- クイズは**サーバー単位**で管理されます（1サーバー=1ゲーム）
- `/new-game` でリセットしないと前のゲームが残ります
- 出題者は自分では回答できません
- お題は**ひらがな・カタカナ**で入力してください

# 👥 VC訪問機能

農場画面の「👥 訪問する」ボタンで、**同じボイスチャンネル**にいるメンバーの農場を見に行けます。

- 相手の農場の外装・内装・庭がそのまま表示されます
- 家の中（室内）まで見られます
- 操作はできません（見るだけ）
